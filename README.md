Launch Instructions
1. Install Python
Download from python.org

During installation, check the box "Add Python to PATH"

2. Install Ollama
Download from ollama.com/download

Install (simply click "Next")

3. Download Mistral model
Open PowerShell

Run: ollama pull mistral

Wait for download to complete (~4 GB)

4. Install requests library
In PowerShell: pip install requests

5. Run the code
select the folder of code in terminal
type ''python financial_local.py'' in terminal
